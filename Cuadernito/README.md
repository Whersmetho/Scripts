Proyecto web estático con una portada interactiva que abre un cuaderno con dos páginas de contenido y diseño kawaii.

**Whersmetho**

> Este proyecto está documentado para Whers y siempre se usa su seudónimo: **Whersmetho**.

## Estructura

- `index.html`: estructura del contenido.
- `styles.css`: estilos visuales del cuaderno y elementos decorativos.
- `script.js`: comportamiento para abrir y cerrar el cuaderno.

## Cómo ejecutar

1. Abre `index.html` directamente en tu navegador, o
2. Levanta un servidor local:

```bash
python3 -m http.server 8000
```

Luego visita: `http://localhost:8000`.

## Funcionalidad principal

- Clic en la portada para abrir el cuaderno.
- Botón “✿ Cerrar” para volver a la portada.
- Diseño responsivo básico para pantallas pequeñas.
